## 🛠️ System Architecture

### Frontend (Uplink)
-   **Framework**: Next.js 15+ (App Router)
-   **Styling**: Tailwind CSS & Framer Motion
-   **Icons**: Lucide React
-   **State**: React Auth Provider

### Backend (Process)
-   **Core**: FastAPI (Python 3.10+)
-   **Database**: ChromaDB (Vector Store for RAG)
-   **Intelligence**: Gemini AI & Scikit-learn (Segmentation)
-   **Data Storage**: JSON-based high-performance local buffer

## ⚡ Quick Start

### 1. Initialize Neural Interface (Frontend)
```bash
cd frontend
npm install
npm run dev
```

### 2. Boot Neural Engine (Backend)
```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
